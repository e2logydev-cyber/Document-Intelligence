Complete Data Sequence
Step From → To Action Data Payload
1Upload
 Blazor → Blob User uploads RFP PDF via drag-drop
POST /blobs/proposals { file: RFP.pdf, metadata: {uploadedBy, timestamp} }
2Trigger
 Blazor → Foundry Initialize agent session with blob URL
POST /agents/sessions { blobUrl: "...", userId: "...", proposalId: "..." }
3Parse
 Foundry → Doc Intelligence Extract structured data from RFP (MCP tool)
MCP: documentIntelligence.analyze(blobUrl) → {clientName, scope, requirements[], budget, deadline}
4Search
 Foundry → MS SQL + AI Search Hybrid vector search for relevant content
SELECT TOP 5 * FROM PastProjects ORDER BY CosineSimilarity(@embedding, ProjectVector); + AI Search queries
5Assemble
 Foundry IQ ↻ Internal Rank, deduplicate, and assemble context
Foundry IQ: iterative reflection, confidence scoring, provenance tracking
6Generate
 Foundry → Azure OpenAI GPT-4o generates 5 proposal sections
model.generate({context, schema: ProposalSchema, outputFormat: "structured_json"})
7Store
 Foundry → MS SQL → Blazor Save sections + SignalR push to clients
INSERT ProposalSections + SQL Query Notification → SignalR Hub → Client update
8Review
 Blazor → Foundry Role-based approval workflow
POST /review {sectionId, reviewerRole, decision, comments} → Agent processes revision
9Deliver
 Foundry → Blob + SQL Export PDF/DOCX, update status, notify
MCP: blob.upload(pdfBytes) + UPDATE Proposals SET status='Submitted' + Email notification
